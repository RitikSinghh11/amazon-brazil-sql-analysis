## 📖 Project Overview

This project presents an end-to-end SQL analysis of the **Amazon Brazil E-commerce Dataset** using **PostgreSQL**. The analysis focuses on extracting business insights from customer orders, products, sellers, payments, and reviews through structured SQL queries.

By applying advanced SQL techniques, this project demonstrates how transactional data can be transformed into actionable insights that support business decision-making.

---

## 🎯 Business Problem

E-commerce platforms generate millions of transactions every day. Without proper analysis, valuable business opportunities remain hidden.

This project aims to answer key business questions related to customer purchasing behavior, product performance, seller efficiency, payment preferences, and sales trends using SQL.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Identify top-performing product categories.
- Evaluate seller performance.
- Analyze payment methods.
- Study sales trends over time.
- Generate business insights using SQL.
